## Amazon Sales Analysis | End-to-End Data Analyst Project
Project Overview

This project is an end-to-end Amazon Sales Analysis project built using Excel, Python, SQL, and Microsoft Power BI. The objective of the project is to clean, analyze, and visualize Amazon e-commerce sales data to generate meaningful business insights and support data-driven decision-making.

Key Business Insights
Maharashtra, Karnataka, and Telangana generated the highest sales revenue.
Product categories such as Set and Kurta contributed the most revenue.
April recorded peak sales performance.
Cancelled and returned orders highlighted operational and logistics challenges.
The dashboard provides insights into customer ordering trends and regional sales performance.
Business Recommendations
Focus marketing efforts on high-performing states.
Improve logistics to reduce cancelled and returned orders.
Increase inventory for high-demand product categories.
Use regional sales trends for targeted business expansion.
