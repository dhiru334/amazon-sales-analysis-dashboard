# Amazon Sales Analysis Dashboard

## Project Overview

This project focuses on analyzing Amazon sales data using Excel and Microsoft Power BI. The objective of the project is to perform data cleaning, exploratory data analysis (EDA), and build an interactive dashboard to generate business insights from raw sales data.



## Tools & Technologies Used

* Excel
* Microsoft Power BI
* Data Cleaning Techniques
* Pivot Tables
* KPI Analysis
* Data Visualization



## Dataset Information

The dataset contains Amazon e-commerce sales records with details such as:

* Order ID
* Product Category
* Sales Amount
* Order Status
* State & City Information
* Shipping Details
* Sales Channel
* Customer Order Data



## Data Cleaning Process

The following data cleaning steps were performed:

* Handled missing values
* Removed unnecessary columns
* Standardized text formatting
* Checked duplicate values
* Corrected date formatting
* Cleaned country and shipping information



## Dashboard Features

The interactive Power BI dashboard includes:

* Total Sales KPI
* Total Orders KPI
* Monthly Sales Trend Analysis
* Sales by State
* Sales by Category
* Order Status Analysis
* Interactive Slicers & Filters



## Key Insights

* Maharashtra, Karnataka, and Telangana generated the highest sales.
* Product categories such as Set and Kurta contributed the most revenue.
* April recorded the highest sales performance.
* Some operational issues such as cancelled and returned orders were identified.
* The dashboard helps understand regional sales performance and customer ordering trends.


## Business Recommendations

* Focus marketing campaigns on high-performing states.
* Improve logistics and delivery operations to reduce cancelled orders.
* Increase inventory for high-selling categories.
* Analyze low-performing regions for business expansion opportunities.


## Project Outcome

This project helped improve skills in:

* Data Cleaning
* Data Analysis
* KPI Development
* Dashboard Design
* Business Storytelling
* Power BI Visualization
